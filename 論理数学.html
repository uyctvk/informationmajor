命題式

命題式は、真偽値を持つ変数であり、論理式の基本的な構成要素である.
(3 < x かつx < 5) ならば3 < xを命題式で記述すると(P ∧ Q) → Pとなり、この場合TもしくはFのどちらかになる.その為命題変数と呼ばれる.

論理演算子

論理演算子は、命題もしくは変数を組み合わせて新しい命題を形成するために使用する.
論理演算式は下記のようなものがある
∧連言（かつ）（掛け算）
//A∧B(AかつB)の時PとQの連言もしくは、両方が成り立つ
∨ 選言（または）（足し算）
//P∨F(PまたはF)の時PとFの選言もしくは、（いずれか片方、両方）が成り立つ
¬ 否定（でない）
//¬ Pまたは〜PもしくはP⊥と下記Pの反対の意味を持つ
//2重否定は元の数字に戻る
→含意（ならば）
//一般的な考え方とは異なるものであり、一般的にはカラスが黒い→地球は丸い(カラスが黒いならば地球は丸い)と解釈し解はF,偽となる.
しかし、数学では因果関係は無い.
//数学的解釈は後述の通りとなる
1,一般的にカラスは黒い.これはT,真となる
2,地球は丸い.これもT,真となる
それらを統合しT→Tとなる式の解は正しいためT,真となる.よって数学的に言えば、この式はTが解となる
↔同値（左の式になるならば右の式になる）
これも含意の時と同様にT↔TなたT,真である
命題式の例を下記に示す
(T∧(P →Q))∨(Q →F)等や(R∨(T∧(P∧Q)))等がある.

真理値表

真理値表は命題式から真偽を問うものである.
T,真（命題が正しいときに記載する）
F,偽（命題が間違っている時に記載する）
真理値表を作成するときは、N変数の命題→＝2n行で計算する
例えば（P, Q, R）は３変数の命題であり23で８行の真理値表を作成すると全ての真理値を確認することができる.
論理演算子の優先順位は→,↔<∧<∨<¬となっている
連言・選言・含意は右結合的である.という法則が存在する.これは右から順に計算していくというものであり、
下記の例を示す
元の式 括弧を明示した式
A → B → C A → (B → C)
A ∧ B ∧ C A ∧ (B ∧ C)
A ∨ B ∨ C A ∨ (B ∨ C)
P∧Q →R S →T (P∧Q) → ((R∧S) → T)
P∧(Q →R ∧S) →T (P∧ (Q → (R∧S))) → T
P∧(Q →R) S →T (P∧ ((Q → R) S)) → T
ちなみにC言語の除算演算子においては左結合的である

トートロジー(恒真命題)の定義

真理値表の最終結果が全てT,真となる命題の事をトートロジーと呼ぶ
トートロジーの有名なものとして排中律が存在する.排中律の式は、
「Pでない.または、Pである」（¬P ∨ P）であり、この場合トートロジーが成り立つ
トートロジーの有名なものとしてパースの法則がある.これは含意が３つ以上存在し他の論理演算子が含まれない式である.
この場合トートロジーが成り立つ
対偶でのトートロジーも存在する.対偶とは左辺と右辺を交換したものであり、A=BならばB=Aとして、さらに両方を否定した-B=-Aとなる.
対偶のトートロジーは「(P → Q) ⇔ (¬Q → ¬P)」であり、この場合トートロジーである.

論理式

量子化

全称量化子（for all）∀「すべての」「任意の」と呼ぶ
例の論理式を上げる.∀x (x > 0 → x2 > 0)　すべてのXはX＞０ならばX2
存在量化子（there exists）∃「ある」「存在する」
意味として、少なくとも1つの対象について命題が成り立つ
∃x (x2 = 4)　「x2 = 4 となるようなxが少なくとも1つ存在する」
量子化の否定は¬∀x P(x) ⇔ ∃x ¬P(x) （全称の否定 → 存在に変わる）
¬∃x P(x) ⇔ ∀x ¬P(x) （存在の否定 → 全称に変わる）
集合名 記号 説明
自然数 ℕ x ∈ ℕ は x が自然数
整数 ℤ x ∈ ℤ は x が整数
有理数 ℚ x ∈ ℚ は x が有理数
実数 ℝ x ∈ ℝ は x が実数
複素数 ℂ x ∈ ℂ は x が複素数（例：2i+3 など）
∀x∈A.P　すべてのAの要素xについてPが成り立つ
∀x∈ ℝ.x2 ≥0は真、∀x∈ ℝ.x>0は偽=-1の時に正しくない
∃x∈A.P　あるAの要素xについてPが成り立つ

酔っ払いの定理（The Drinker Paradox）

任意の集合 S の中で、命題 D(x)（xは飲んでいる）に対して：
∃x∈S, D(x)→∀y∈S,D(y)
つまり：「あるxがいて、xが飲んでいるならば、すべてのyが飲んでいる」
この命題は常に真（トートロジー）になります.
